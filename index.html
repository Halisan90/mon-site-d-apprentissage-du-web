Concepts clés

Partie 1 - Découvrez le web et ses origines

Web

Le web est un service d'internet, il permet de consulter des pages grace aux liens hypertext.

Internet

C’est le réseau qui permet de communiquer entre ordinateurs

Cloud

Le cloud permet de conserver des données et des services

W3C

Le W3C c'est le WEB tel que nous le connaissons aujourd'hui

Partie 2 - Distinguez les langages du Web

Langages client

Un langage côté client est un langage qui fonctionne dans le navigateur de l’utilisateur pour afficher et rendre la page web interactive, comme HTML, CSS et JavaScript.

Navigateurs

Le navigateur à pour rôle de traduire les langages HTML, CSS et JavaScript sous la forme de sites web utilisables par tout le monde.

Langages serveur

Un langage côté serveur sert à faire fonctionner un site web et gérer les données sur l’ordinateur qui héberge le site.

Frameworks

Un framework est un ensemble d’outils et de règles qui aide les développeurs à créer plus facilement des applications ou sites web, sans tout coder depuis zéro.

Bases de données

Une base de données est un endroit où on garde toutes les informations d’un site ou d’une application.

Site responsive

Un site responsive est un site qui s'adapte à la taille de tout les écrans pc,tablette ou smartphone.

Application mobile

Une application mobile est un logiciel specialement conçu pour les mobile et tablette.

Partie 3 - Découvrez le fonctionnement du réseau

Serveurs

Un serveur est un ordinateur qui stocke des sites ou applications et envoie les informations aux utilisateurs quand ils y accèdent.

Adresse IP

L’adresse IP permet de retrouver un ordinateur sur la planète. Chaque ordinateur a sa propre adresse IP.

Nom d’hôte et DNS

Le nom d'hôte est le nom du domaine afin de retrouver plus facilement la pages rechercher et le DNS est l'annuaire qui traduit l'adresse IP en nom d'hote.

Protocoles

Les protocoles sont des langages de communication pour que les machines se parlent entre elles.

Partie 4 - Identifiez les métiers du web

Développeurs

Un développeur est une personne qui crée des sites, applications ou logiciels en écrivant du code pour les faire fonctionner.

Product Managers et chefs de projets

Le product Manager définit ce que le produit doit être. Il planifie la vision du produit. Le chef de projet s’assure que le projet avance bien et se termine correctement.

Designers

Un designer est une personne qui conçoit l’apparence et l’expérience d’un site ou d’une application pour qu’il soit joli et facile à utiliser.

Testeurs

Un testeur est une personne qui vérifie qu’un site ou une application fonctionne correctement et signale les problèmes à corriger.

Techniciens informatique

Un technicien informatique est une personne qui installe, répare et maintient les ordinateurs, réseaux et systèmes informatiques pour qu’ils fonctionnent correctement.

Langages de développement web

HTML

Front-End

En front-end, HTML sert à structurer le contenu d’une page web, en définissant les titres, paragraphes, images, liens, formulaires et autres éléments.

CSS

Front-End

En front-end, CSS sert à styliser et mettre en forme les pages web, en définissant les couleurs, polices, tailles, espacements, mises en page et animations.

JavaScript

Front-End

En front-end, JavaScript sert à rendre les pages web interactives et dynamiques, en manipulant le HTML, le CSS et en réagissant aux actions des utilisateurs.

JavaScript

Back-End

En back-end, JavaScript sert à exécuter du code côté serveur, gérer les requêtes/réponses, accéder aux bases de données et construire la logique d’une application web.

PHP

Back-End

En back-end, PHP sert principalement à générer des pages web dynamiques, traiter des formulaires, gérer des sessions/utilisateurs et interagir avec des bases de données.

Java

Back-End

En back-end, Java sert à développer des applications robustes et sécurisées, souvent pour gérer la logique métier, interagir avec les bases de données et assurer la communication entre le serveur et les clients.

Python

Back-End

En back-end, Python sert à développer la logique des applications, créer des API, interagir avec des bases de données, automatiser des tâches et construire des sites ou services web (souvent avec des frameworks comme Django ou Flask).

C#

Back-End

En back-end, C# sert à développer des applications web, services et API, souvent avec le framework .NET, pour gérer la logique métier, communiquer avec les bases de données et assurer la communication serveur-client.

SQL

Bases de données

En back-end, SQL sert à interagir avec les bases de données, c’est-à-dire créer, lire, mettre à jour et supprimer des données
